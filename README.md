# 1000X — Agency Portfolio

A single-page portfolio for **1000XDEV / ThousandX Ventures LLP**, an AI-native product studio.

- **Project-first layout** — brief "who we are", then 26 shipped products, flagships leading.
- Each project card shows a **live screenshot** + a crisp description and links to the real product.
- Category filters (AI · EdTech · E-commerce · Fintech · Mobile · Enterprise), scroll-reveal animation, dark AI-style theme.

## Run

Open `1000x-portfolio.html` in any browser. Screenshots live in `images/` — keep that folder next to the HTML.

## Deploy (GitHub Pages)

Settings → Pages → deploy from `main` branch, then rename `1000x-portfolio.html` to `index.html` (or add a redirect).
